## What's live now

    Account (App Store reviewers read these)
- **`delete_my_account()` RPC** (migration 00048) — anonymizes profile
  (username → `deleted_<uuid>`, clears display_name/avatar/bio), sets
  `deleted_at` timestamp, locks `auth.users` row
  (`banned_until = infinity`, clears email/phone/password), marks active
  subscriptions `cancel_at_period_end = true`
- **RLS update**: `profiles` SELECT policy now filters tombstoned rows
  from public reads
- **Auth hook safety net**: any session that lands on a tombstoned
  profile is force-signed-out immediately

## Deferred — architectural decisions

- **Per-page OG tags** for memorial sharing on social: Vercel Edge
  middleware that intercepts crawler User-Agents, fetches memorial
  data from Supabase, returns HTML with inlined OG tags before the
  SPA hydrates. ~60 LOC but needs a deploy-time decision.
- **Hard-delete cron** for deleted accounts (currently soft-delete
  only): a scheduled edge function that purges
  `profiles.deleted_at < now() - interval '30 days'` would close
  GDPR's true right-to-erasure. Requires also fixing the 25 bare
  `REFERENCES profiles(id)` columns identified earlier
  (memory_vaults, virtual_spaces, family_trees, etc.).
- **Test coverage for React Query hooks**: would need
  `@testing-library/react` + `@testing-library/react-hooks` +
  per-test QueryClient setup. Worth it once test suite > 100 tests.
